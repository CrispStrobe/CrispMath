the `symbolic_math_bridge` plugin and the web build's `symengine.wasm`.

────────────────────────────────────────────────────────────────────────────
LGPL compliance — source, build, and relink materials
────────────────────────────────────────────────────────────────────────────
CrispMath distributes prebuilt binaries that include GMP, MPFR, MPC, and FLINT,
which are distributed under LGPL-family licenses as noted above. On native
platforms these libraries are statically linked through the `symbolic_math_bridge`
plugin. On web they are linked into `web/symengine.wasm`.

The application source code is available at:
  https://github.com/CrispStrobe/CrispMath

The Flutter bridge and bundled platform binaries are built from:
  https://github.com/CrispStrobe/symbolic_math_bridge

The native Apple math-stack build scripts and pinned dependency versions are at:
  https://github.com/CrispStrobe/math-stack-ios-builder

Those repositories are the preferred source/build route for rebuilding the app
or math stack with modified LGPL libraries. For distributed App Store/mobile
binaries where the published source is not enough to relink the exact released
binary, you may request the corresponding object files or equivalent relink
materials for that release by contacting:
  support@crispstro.be

This offer is valid for at least three years after the corresponding binary
release. The LGPL library sources are available at the upstream URLs listed
above.
